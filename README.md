# 5分間通話アプリ

見知らぬユーザーとランダムマッチングして、5分間だけ音声通話できるアプリです。

## セットアップ

### 必要なもの
- Node.js 16以上

### 起動手順

```bash
# 依存パッケージをインストール
npm install

# サーバー起動
npm start
```

ブラウザで http://localhost:3000 を開きます。

## テスト方法

同じPCで試す場合：
1. ブラウザのタブを2つ開く（シークレットウィンドウを使うと確実）
2. 両方で「通話を始める」をクリック
3. マイクの許可を与えるとマッチングして通話が始まります

別のPCやスマホで試す場合：
1. 同じWi-Fiに接続する
2. サーバーを起動しているPCのIPアドレスを確認（例: 192.168.1.10）
3. `http://192.168.1.10:3000` にアクセス

## ファイル構成

```
5min-call/
├── server.js        # マッチング + シグナリングサーバー
├── package.json
└── public/
    └── index.html   # フロントエンド（UI + WebRTC）
```

## 仕組み

1. **マッチング**: Socket.io で待機キューを管理。2人揃ったらペアリング
2. **シグナリング**: WebRTC の接続情報（SDP・ICE）をサーバー経由で交換
3. **通話**: P2P（WebRTC）で直接音声を送受信。サーバーは中継しない
4. **タイマー**: 5分経過でサーバーとクライアント両方から終了通知

## クラウドにデプロイする場合

Railway や Render にデプロイする際は、WebSocketが使えるプランを選んでください。
HTTPS環境では WebRTC が正常に動作します（ローカルは http://localhost でOK）。
